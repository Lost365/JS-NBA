Este proyecto llamado "NBA Stats Viewer" es una aplicación web desarrollada en HTML, CSS y JavaScript que consume la API pública de la NBA a través de RapidAPI para mostrar estadísticas individuales de jugadores en partidos específicos; actualmente está en desarrollo pero funcional en entorno local, y se planea agregar filtros por equipo, selección de partidos y visualización gráfica; utiliza tecnologías como HTML5, CSS3 y JavaScript ES6 sin librerías externas, y requiere una clave válida de RapidAPI para funcionar; para instalarlo basta con clonar el repositorio, abrir el archivo index.html en el navegador y reemplazar la clave en el archivo script.js; se han detectado algunos bugs como partidos sin datos, falta de validación visual y ausencia de logos de equipos; entre las preguntas frecuentes se incluye cómo obtener un gameId válido, si se puede filtrar por jugador (aún no), y por qué no se muestran todos los jugadores (solo los que tienen estadísticas registradas); el proyecto está bajo licencia MIT y puede ser modificado libremente; el equipo está conformado por Sebastián Castrillón Cetre como desarrollador frontend, encargado de la integración de la API y el diseño visual.
